# 최초 연결법
- usb를 연결한다.
- 이 맥에 관하여 >  시스템리포트 > USB > Wireless Controller 가 보이는지 체크
- 맥의 블루투스를 켜고 설정창을 띄운다.
- share + ps 버튼을 동시에 누르면 고속으로 깜빡인다.
- 페어링 한다.
